| Package           | Description                                                      |
| ----------------- | ---------------------------------------------------------------- |
| `@confect/core`   | Shared specs and schemas used by all Confect packages            |
| `@confect/cli`    | Developer tooling for codegen and sync                           |
| `@confect/server` | Backend bindings to the Convex platform                          |
| `@confect/react`  | Client-side bindings for React apps                              |
| `@confect/test`   | Utilities for testing Confect apps without a live Convex backend |
